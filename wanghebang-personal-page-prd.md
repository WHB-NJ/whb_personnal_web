# 王赫邦个人主页 - 产品需求文档

## 1. 产品概述

### 1.1 产品定位
打造一个兼具苹果公司高级感、文艺气质与时代先锋特征的极致个人网页，全方位展示王赫邦在药学数智医疗、AI创新、骑行探险等领域的多维成就。

### 1.2 目标受众
- 高校学术圈与科研合作方
- 医疗科技行业从业者与投资人
- 户外运动与骑行爱好者社群
- 对AI与前沿科技感兴趣的青年群体

### 1.3 核心价值主张
一个跨学科的先锋探索者——药学与AI的交叉实践者 + 极限骑行的组织者 + 时代前沿的思考者。

## 2. 品牌调性

### 2.1 视觉风格
- **核心理念**：Apple级极简主义 + 文艺先锋的克制表达
- **关键词**：呼吸感、留白、精确、克制、先锋、温度
- **拒绝**：过度装饰、花哨渐变、廉价动画、堆砌元素

### 2.2 字体策略
- **中文显示**：优先使用系统级优质字体（PingFang SC, Noto Sans SC）
- **英文显示**：选用独特而优雅的字体组合，避免使用Inter/Roboto/Arial等泛滥字体
- **字重运用**：大胆使用超粗与超细字重对比，营造视觉张力

### 2.3 色彩系统
- **主色调**：深邃黑 (#0a0a0a) 与纯白 (#ffffff) 的极致对比
- **辅助色**：温暖的灰调层次（#1a1a1a, #2a2a2a, #666666, #999999）
- **点缀色**：一个精心挑选的先锋色彩（建议采用医疗蓝绿或骑行山野绿），用于关键交互和视觉锚点

### 2.4 排版原则
- 大留白、大字号、强烈的层级对比
- 非对称布局与黄金比例的结合
- 文字作为视觉元素本身，而非单纯信息载体

## 4. 交互与动效

### 4.1 页面加载
- 优雅的加载过渡效果
- 首屏内容分段渐入

### 4.2 滚动体验
- 基于IntersectionObserver的视差滚动
- 文字逐行/逐段渐显
- 图片优雅入场（淡入 + 微位移）

### 4.3 微交互
- 链接hover状态的精致变化
- 图片hover的微妙反馈
- 导航锚点的平滑滚动

### 4.4 响应式
- 完美适配桌面端与移动端
- 移动端优先的触摸友好交互

## 5. 技术约束

### 5.1 技术栈
- 纯HTML + CSS + JavaScript，无外部框架依赖
- 使用CSS变量维护设计系统一致性
- 所有动画使用CSS原生方案或轻量JS

### 5.2 性能要求
- 首屏加载时间 < 2秒
- 动画帧率稳定在60fps
- 图片使用合适尺寸与格式

### 5.3 资源利用
- 现有图片素材直接使用（珠峰AI海报、GAIE展览图）
- 视频元素（Bilibili宣传片）提供播放入口

## 6. 成功标准

### 6.1 视觉质量
- 每个像素都经过深思熟虑
- 整体质感达到Apple官网级别
- 文艺气息与先锋感的平衡

### 6.2 用户体验
- 信息层级清晰，阅读流畅
- 交互自然，无明显延迟
- 移动端体验同样精致

### 6.3 个人品牌
- 访客能立即感知到这是一个"不简单的人"
- 跨学科特质的自然呈现
- 留下深刻印象，愿意进一步了解
